# Jens-event-talks-app

This repository hosts the static website for a 1-day event filled with technical talks.

## Features:
- Schedule for 6 technical talks and a lunch break.
- Talk details include title, speakers, categories, duration, and description.
- Search functionality to filter talks by category.

## How to run locally:
1. Save the `index.html` file to your local machine.
2. Open your terminal in the same directory and run `python3 -m http.server 8000` (or `python -m http.server 8000`).
3. Open your web browser and navigate to `http://localhost:8000`.
